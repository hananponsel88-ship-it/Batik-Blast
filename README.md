# Batik-Blast
Batik Blast adalah game mobile edukasi berbasis puzzle yang mengajak pemain menyusun motif batik pada papan grid dengan tiga tingkat kesulitan. Dilengkapi fitur Store, Batik Collection, dan Leaderboard, game ini hadir sebagai sarana hiburan sekaligus media pelestarian budaya batik Nusantara berbasis platform digital.
